Delete all tag values Beta

Permanently delete every option from a tag or select field — removes values from the field's option list and unsets them from every entity that has them assigned.
How to use
  • Select a field — choose the tag or select field to clean up. Only UUID-keyed custom fields are listed (Tags, MultiSelect, SingleSelect). Status fields are not supported.
  • Confirm — type DELETE to enable the button.
  • Effect — every value is removed from the field's option list. Any entity that had the value assigned will have it silently unset (force=true).
  • No undo — deleted values cannot be recovered. Export your data before running if needed.
⛔ This permanently removes every value from the selected field and unsets them from all entities. There is no undo.

Delete tag values from CSV Beta

Delete specific field values by name — upload a CSV with the values you want to remove. Only values that match a CSV row are deleted.
How to use
  • CSV format — one tag value name per row in a single column. A header row is expected (e.g. tag_name). Extra columns are ignored.
  • Matching — case-insensitive, trimmed. Values not found in the field are silently skipped.
  • Effect — matched values are removed from the field's option list and unset from all entities that had them assigned (force=true).

Diff by CSV Beta

Upload a CSV with the values you want to keep — everything else is deleted. Useful for pruning a field down to a canonical list.
How to use
  • CSV format — one tag value name per row. Values listed in the CSV are kept. All others are deleted.
  • Matching — case-insensitive, trimmed. Values in the CSV that don't exist in the field are silently ignored.
  • Effect — unmatched values are removed from the field's option list and unset from all entities that had them assigned (force=true).
  • No undo — export your data first if you need a rollback option.

Delete by selecting Beta

Load all values for a field and pick which ones to delete using a checklist.
How to use
  • Select a field — choose which tag or select field to manage.
  • Load values — fetches all current values from Productboard.
  • Select — check values to delete. Use Select all / Invert / Deselect all to bulk-manage.
  • Delete selected — selected values are removed from the field and unset from all assigned entities (force=true).
